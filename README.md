# StudySphere AI

## Overview

StudySphere AI is a smart learning platform built using Python and Flask. It helps students organize their studies, track progress, generate quizzes, and solve academic doubts through a simple and interactive interface.

## Features

* Dashboard with study statistics
* Study Planner
* Quiz Generator
* Doubt Solver
* Progress Tracking
* Achievement System

## Tech Stack

* Python
* Flask
* HTML
* CSS

## Project Structure

AI-Study-Mentor/

* app.py
* templates/
* static/
* requirements.txt
* README.md

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
